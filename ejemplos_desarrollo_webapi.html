Ejemplos de Desarrollo de una Web API

Ejemplo 1: Registro de Usuarios

Desarrolla una Web API que permita a los usuarios registrarse, iniciar sesión y gestionar su perfil.

  • Endpoints:
    • POST /usuarios: Registra un nuevo usuario.
    • POST /login: Inicia sesión de usuario.
    • GET /usuarios/{id}: Obtiene detalles de un usuario.
    • PUT /usuarios/{id}: Actualiza información de usuario.
    • DELETE /usuarios/{id}: Elimina un usuario.
  • Ejemplo de petición: 
    POST /usuarios
{
  "nombre": "Ejemplo",
  "email": "ejemplo@example.com",
  "contraseña": "secreta123"
}
Ejemplo 2: Gestión de Productos

Crea una Web API para administrar un catálogo de productos, incluyendo operaciones de crear, leer, actualizar y eliminar productos.

  • Endpoints:
    • GET /productos: Obtiene todos los productos.
    • GET /productos/{id}: Obtiene detalles de un producto.
    • POST /productos: Crea un nuevo producto.
    • PUT /productos/{id}: Actualiza información de un producto.
    • DELETE /productos/{id}: Elimina un producto.
  • Ejemplo de petición: 
    POST /productos
{
  "nombre": "Producto Ejemplo",
  "precio": 99.99,
  "cantidad": 10
}
